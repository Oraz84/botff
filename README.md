# Telegram GPT-5 + Google Drive Bot (Render.com)

## Deployment
1. Upload project to GitHub
2. Create Render Web Service:
   - Build: `pip install -r requirements.txt`
   - Start: `python bot.py`

## ENV variables:
- TELEGRAM_BOT_TOKEN
- OPENAI_API_KEY
- GOOGLE_DRIVE_FOLDER_ID
- GOOGLE_SERVICE_ACCOUNT (json string)
